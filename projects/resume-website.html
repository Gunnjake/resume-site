Portfolio & Resume Website

Overview

A clean, minimal, custom-built portfolio website designed to host my resume, experience, and project pages.

Key Features

  • Deployed using GitHub Pages with a custom domain.
  • Fully static front-end using HTML, CSS, and lightweight JavaScript.
  • Modular, scalable folder structure for maintainability.
  • Individual detail pages for each experience and project.

Technologies Used

HTML, CSS, JavaScript, GitHub Pages
